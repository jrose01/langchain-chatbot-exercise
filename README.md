# langchain-chatbot-exercise
This repo stores an exercise for QAC387 that introduces LangChain LCEL and few-shot prompting

## Exercise: Break the BOT

The purpose of this exercise is learn how to make API calls, practice prompting, experimenting with tone, and observing response output. You do not need to know the coding.

Step 1: Open Github Desktop and clone the Github repository to a local Git repository from the web url: 

https://github.com/jrose01/langchain-chatbot-exercise.git

Create a Workspace for this folder in VS Code and download the .env file located on the Moodle web site to the workspace folder.

Create and activate a virtual environment.

Run the requirements.txt file: pip install -r requirements.txt

----------------------------------------------------------------

Open the LCEL_chatbot_example.py and follow the instructions

### Instructions

1) Prompt the chat bot for answers to a few math questions.
2) Change a few of the examples using the exercise to see how the chatbot responds.

For fun:  
Customize the chatbot for different roles, such as:  

- A math assistant focused on calculations and numerical queries.  
- A casual chatbot for fun interactions.  
- A math assistant that is not good at math.  

Test the chatbot with various questions to see how it responds based on its role and examples.
